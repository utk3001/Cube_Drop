# Cube_Drop
Another 2-D arcade game built using unity and c#.<br/>
The player in a form of cube can move sideways.<br/>
There are different types of platforms moving upwards and we need to land from platform to platform.<br/>
Game gets over if the cube falls down or gets in contact with a spike.<br/>
The scoring system is time based.<br/>
Game is also integrated with sound effects.<br/>
Jump Mechanism to be added in the future.<br/>
